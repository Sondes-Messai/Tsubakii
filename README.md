# 🌸 Tsubakii 🌸

Tsubakii🎋 est une application mobile📱 innovante utilisant l'intelligence artificielle🧠 pour reconnaître les plantes grâce à la caméra de votre appareil📽️. En plus d'identifier les plantes, l'application génère des citations inspirantes liées à chaque plante reconnue.

## 🌟 Fonctionnalités 🌟

- **🔍 Reconnaissance des plantes** : Utilisation de la caméra AI pour identifier les plantes.
- **💬 Génération de citations** : Génération de citations inspirantes pour chaque plante reconnue.
- **📱 Interactivité** : Interface utilisateur simple et intuitive pour prendre des photos et voir les résultats.

## 🔧 Technologies Utilisées 🔧

- **Frontend** : 
  - React Native 📱
  - JavaScript 💻
  - Expo 🚀

- **Backend** : 
  - .NET Core 🔧
  - API REST 🌐

- **Base de Données** : 
  - SQL Server 🗄
  - Entity Framework Core 🛠

- **Infrastructure** : 
  - Docker 🐳

## 🚀 Prérequis 🚀

- Node.js (version 16 ou supérieure) ⚙️
- Expo CLI 🔧
- .NET SDK 🛠
- Docker 🐳

## 📥 Installation 📥

1. **Cloner le dépôt** :
   ```bash
   git clone https://github.com/ton-utilisateur/tsubakii.git
   cd tsubakii
   ```

2. **Installer les dépendances** :
   ```bash
   npm install
   ```

3. **Configurer l'environnement** :
   - Créez un fichier `.env` à la racine du projet et ajoutez vos clés API et configurations nécessaires. 🗝

4. **Lancer l'application** :
   ```bash
   npx expo start
   ```

5. **Configurer le Backend** :
   - Naviguez vers le répertoire `backend` et démarrez le serveur .NET Core :
     ```bash
     cd backend
     dotnet run
     ```

6. **Utiliser Docker** :
   - Construire et démarrer les conteneurs Docker :
     ```bash
     docker-compose up --build
     ```

## 📝 Utilisation 📝

1. **Démarrer l'application Expo** :
   - 📲 Scannez le QR code généré avec l'application Expo Go sur votre appareil mobile pour lancer l'application.

2. **Prendre une photo** :
   - 📸 Utilisez la caméra intégrée pour capturer une image de la plante.

3. **Recevoir l'identification de la plante et la citation** :
   - 🌱 L'application enverra l'image à l'API pour l'identification et affichera la plante reconnue ainsi qu'une citation inspirante générée par l'IA. Par exemple :
     - 🌿 **Plante reconnue** : Camellia Japonica (Tsubaki)
     - 💬 **Citation générée** : "Comme le tsubaki qui fleurit sous la neige, la beauté naît dans la simplicité et la résilience."
